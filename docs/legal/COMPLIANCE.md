# Compliance Overview

This file summarizes legal & compliance requirements for Verse Guy (excerpted from the Ultimate Specification).

## Core obligations
- GDPR, CCPA and DSA compliance must be built-in.
- Per-country laws must be considered (UK, AU, JP, BR, etc.).
- ToS acceptance is required for CIG ToS enforcement.
- Audit logging (append-only, tamper-evident) must be provided for legal requests.

## Implementation notes
- Region detection and consent banners for EU requirements.
- Data export & deletion APIs must be available for user rights.
- Data minimization and privacy-by-default principles apply.

See `legal/ToS.md`, `legal/PrivacyPolicy.md` and `docs/extracted/gdpr.md` for details.
